


Has anyone tested google's stitch ai design tool yet?

I read about it and started to test it out. I am not a designer, so please forgive my not so sophisticated opinion on it. But I have some basic knowledge in figma, which until now, seemed to me the one and only tool for designs.

But coming back to what I, as non designer, in just a few clicks was able to do is quite mind blowing. 

so, what did i do:
my plan was to create a design for a simple image slider page. Thats it. nothing more fancy. as i just wanted to try it out first. so, I created an api key on stitchs settings page and set up the mcp server client in vs code so i could connect to stitch server. using the vscode chat i prompted the command to build this design. 
in my stitch google account then i could review my design. this design can be shared via qr code for others to see on their devices. i then exported the design using the stitches ai studio. it created html files based on this design. 

all in all it took me from setting up stitch, creating the design and translating it into html files not more thamn 20 minutes. 

of course i havent tried out everything and of course everyone needs to  consider for themselves if they have any use of it. however, it was a very very interesting thing for me to try out and it seems to me that figma got a real competition here.
